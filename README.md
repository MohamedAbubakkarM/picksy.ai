# Picksy.ai - Smart Deal Finder

## Overview
DealFinder Agent is a full-stack application designed to help users discover the best deals available across popular e-commerce platforms such as Amazon, Flipkart, and Myntra. The app aggregates pricing data, provides AI-based recommendations, and delivers actionable insights, making online shopping more informed, cost-efficient, and enjoyable.

## Features
- **Best Deal Highlighting**: Clearly marks the best available deal for a given product.
- **Detailed Analysis**: Provides shipping advantages, discount benefits, and trusted seller information.
- **Personalized Recommendations**: Suggests the best products and platforms for purchases.
- **Modern Stack**: Created using Next.js, TailwindCSS, and CrewAI for intelligent deal finding.
- **Search and Compare**: Enables quick searches across platforms and compares pricing and specifications.

## Architecture
Picksy.ai consists of both a frontend and a backend, making it more developer-friendly. The frontend, built with Next.js and TailwindCSS, delivers a seamless user experience, while the backend, powered by Python (FastAPI), CrewAI, and Tavily, handles intelligent deal-finding and data aggregation, highlighting the distinct roles and technologies used for each part of the application.

## Technologies Used 🖥️🎨🐍🗄️☁️
- **Frontend**: Next.js, TailwindCSS, Framer Motion, Lucide Icons
- **Backend**: FastAPI, CrewAI, Tavily

## Usage
Enter a product name in the search bar and click "Search." The app will display available deals, highlighting the best option, making recommendations, and providing an in-depth pricing and shipping analysis.
